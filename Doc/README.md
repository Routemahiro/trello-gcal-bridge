# Doc

このフォルダは `trello-gcal-bridge` の開発者向けドキュメントをまとめる場所です。  
Cloudflare Workers で Trello の Webhook を受けて、GAS Webアプリに転送する最小構成の橋渡しです。

## 目次

- `overview.md` : システム全体像とデータフロー
- `api.md` : 受け口のAPI仕様（HTTP）
- `setup.md` : 開発環境セットアップとローカル確認
- `operations.md` : デプロイと運用
- `testing.md` : テスト方針と現在の注意点
- `security.md` : 秘密情報とセキュリティ注意
